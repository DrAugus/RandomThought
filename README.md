# [RandomThought](https://draugus.github.io/RandomThought/)

## Develop

prerequisites：

- nodejs(Version >= 16)
- yarn(Version >= 1) (`npm install -g yarn`)

git clone：

```git
git clone https://github.com/DrAugus/RandomThought.git
```

run：

```yarn
yarn install
yarn docs:dev
```

## Contribute

Before you commit, please run `vuepress-vite build docs`
